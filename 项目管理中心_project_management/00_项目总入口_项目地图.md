# 项目总入口_项目地图

## 当前定位

本目录是专科知识图谱项目的软件化管理入口。后续换电脑、换账号、交给同事或开发管理系统，优先读取本目录。

## 核心入口

| 文件 | 用途 |
|---|---|
| `01_项目运行清单_manifest.json` | 项目基础配置、当前治理轮次、最小复制范围 |
| `02_关键文件清单_file_index.csv` | 关键文件与目录在哪里、是否必须复制 |
| `03_目录用途清单_directory_index.csv` | 根目录各文件夹用途、迁移和Git策略 |
| `04_批次登记台账_batch_ledger.csv` | 批次台账入口，本轮先占位，后续由正式台账同步 |
| `05_全局错误指纹索引_error_fingerprint.csv` | 错误防复发索引入口，本轮生成初版 |
| `06_脚本资产台账_script_inventory.csv` | 脚本资产台账副本，识别D类/极高风险脚本 |
| `09_迁移复制清单_copy_checklist.md` | 换电脑/新环境复制哪些文件 |
| `14_心血管内科月度建设计划_202607.md` | 2026-07 心血管内科重点疾病图谱建设目标和顺序 |
