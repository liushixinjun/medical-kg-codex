# AMI 推荐来源裁决样板

本样板用于说明“多份指南共同支持同一 CDSS 推荐时，医生端到底以哪份指南作为主依据”。它不直接写入 Neo4j，必须在正式批次中回到原文证据、页码、推荐等级和证据等级后，再生成可入库数据。

样板结论：

1. AMI 诊断标准优先采用国际心肌梗死通用定义作为主依据，教材作为骨架支持。
2. STEMI 再灌注、直接 PCI、溶栓治疗和溶栓禁忌证需要按具体临床问题分别裁决，不能用“急性心肌梗死整体参考某指南”一把盖住。
3. 抗血小板治疗涉及 STEMI、NSTEMI、出血风险和用药禁忌，当前样板标记为“冲突待裁决”，不能直接进入正式 CDSS 推荐区。
4. 未来新指南发布时，不重建 AMI 疾病骨架；只新增来源、重抽受影响证据、重算来源裁决，并将旧推荐标记为“已过时保留”或“仅知识展示”。

医生端默认展示：

- 最终推荐；
- 适用场景；
- 主依据指南；
- 主证据摘要；
- 推荐等级和证据等级；
- 禁忌或阻断条件。

支持依据和冲突依据放在展开层，用于质控和专家复核。
