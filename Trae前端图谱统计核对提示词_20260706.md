## 首页统计推荐 Cypher

### 1. 疾病大类数量

```cypher
MATCH (n:KGNode {entityType:'DiseaseCategory'})
RETURN count(n) AS disease_category_count;
```

期望值：`12`

### 2. 疾病/专病数量

```cypher
MATCH (n:KGNode {entityType:'Disease'})
RETURN count(n) AS disease_count;
```

期望值：`92`

### 3. 首页“实体”数量

建议首页“实体”不要包含 Evidence，否则会被证据节点淹没。推荐使用：

```cypher
MATCH (n:KGNode)
WHERE coalesce(n.entityType,'') <> 'Evidence'
RETURN count(n) AS visual_entity_count;
```

期望值：`1145`

如前端确实要展示所有 KGNode 总节点，请字段名改成“总节点”，查询：

```cypher
MATCH (n:KGNode)
RETURN count(n) AS kg_node_count;
```

期望值：`34292`

### 4. 关系数量

```cypher
MATCH ()-[r]->()
RETURN count(r) AS relationship_count;
```

期望值：`99269`
