# 附录D：实体归一、别名与术语字典

更新时间：2026-07-11 23:55:00

## 1. 主名规则

| 类型 | 主名要求 | alias 要求 |
|---|---|---|
| 疾病 | 中文标准名优先 | 英文名、缩写、历史名称进入 alias |
| 药品 | 通用名/标准中文全称优先 | 英文缩写、商品名、中文简称进入 alias |
| 检查 | 临床常用中文名优先 | 英文缩写和设备名进入 alias |
| 手术/操作 | 规范操作名优先 | 缩写和同义叫法进入 alias |

## 2. 禁止示例

| 错误 | 正确 |
|---|---|
| 主名：P2Y12 | 主名：P2Y12受体抑制剂，alias 包含 P2Y12 |
| 主名：EXAM-TTE | 主名：经胸超声心动图，alias 包含 TTE |
| 主名：AMI鉴别 | 主名：主动脉夹层排除，关系/分组说明其属于 AMI 鉴别 |

## 3. 字典位置

