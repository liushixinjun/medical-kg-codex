# 附录B：来源体系与教材骨架

更新时间：2026-07-11 23:55:00

## 1. 总原则

基础权威教材/专科专著搭骨架，指南/共识/路径填血肉。

## 2. 来源优先级

| 层级 | 来源 | 用途 |
|---|---|---|
| L1 | 权威教材、专科专著 | 疾病定义、概述、基础病因机制、典型临床表现、基础诊疗框架 |
| L2 | 国内外指南、共识、路径 | 推荐等级、证据等级、适用人群、禁忌、诊疗路径 |
| L3 | 外部权威网站 | 仅用于教材/指南覆盖不足时补定义或基础说明，必须保留来源 |
| L4 | Trae/其他模型候选 | 只能作为候选，必须回到来源证据后才能入图谱 |

## 3. 教材骨架槽位

| 槽位 | 说明 | 是否基础槽位 |
|---|---|---|
| Definition | 疾病定义/概述 | 是 |
| Etiology | 病因 | 是 |
| Pathophysiology | 发病机制/病理生理 | 是 |
| Symptom | 症状 | 是 |
| Sign | 体征 | 是 |
| Exam/LabTest | 检查/检验 | 是 |
| DiagnosisCriteria | 诊断标准 | 是 |
| DifferentialDiagnosis | 鉴别诊断 | 是 |
| RiskStratification | 分层/分级 | 条件必需 |
| TreatmentPlan | 治疗原则/方案 | 是 |
| FollowUp/Prevention/Prognosis | 随访、预防、预后 | 条件必需 |

## 4. 教材与指南边界

教材可以提供“基础事实”和“治疗原则”，但不得伪造指南推荐等级。教材中如出现指南等级，只能作为原文片段保存，不能等同于指南证据字段。

## 5. 双源校验

DOCX 与 PDF 同源教材必须互相校验。若 DOCX 转换缺页、错行、目录层级异常，应回到 PDF 原文复核。
